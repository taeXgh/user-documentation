# Application Review

This guide explains how to collect applications from your website and review them.

## Step 1: Collect Applications

### Run the Collect Applications Script

1. Locate the **Collect Applications** batch script
2. **Double-click** the script to open it
3. A window will appear indicating it's ready to collect applications
4. Press **Enter** or any key to start the collection process
5. Wait for the script to finish collecting applications
6. A message will confirm that applications have been collected
7. Press **Enter** to close the window

The script will:

- Run automatically in the background
- Gather all new applications submitted through your website
- Organize them in your Applications folder

## Step 2: Review Applications

### Open the Applications Folder

1. Navigate to your **Applications** folder on your computer
2. You'll see folders organized by **job position** (for example: Accountant, Auditor, System Administrator, etc.)
3. Open the folder for the position you want to review

### Review Each Application

For each applicant:

1. **Open the Application Document**
   - You'll see a Word document with the applicant's information
   - Review the following details:
     - **Position Applied For** — Which job they applied for
     - **Application Date** — When they applied
     - **How They Heard About Us** — Where they found the job posting
     - **Other Important Information** — Any additional details they provided

2. **Review the Resume**
   - A resume file is included with each application
   - Open and review their work history and qualifications
   - You can save a copy or print it if needed

3. **Move to the Next Application**
   - Close the current application
   - Open the next resume in the folder
   - Repeat the review process

## Tips for Application Review

- Review applications in the order listed in the folder
- Take notes on strong candidates
- If an applicant isn't a good fit, move on to the next one
- Keep organized by position so you can compare candidates easily
